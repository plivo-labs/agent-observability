---
title: Session Timeline
description: Scrollable per-turn timeline synchronized with the recording. Each turn is a band with STT, LLM, and TTS segments; hover to scrub, click to jump.
---

## Usage

```tsx
import { SessionTimeline } from '@/components/session-timeline/session-timeline'

<SessionTimeline />

// Custom turn-click handling (e.g. scroll a sibling panel)
<SessionTimeline onTurnClick={(n) => focusTurn(n)} />
```
